Metalearning!

15-05-2026

Metalearning is one of the 9 principles of Ultralearning.

The idea is that before you learn a subject you should learn how to learn it.

I want to get better at HTML + CSS so I can make a nicer part of my website that can be useful instead of what my blog is which is more often than not raw thoughts in raw HTML.

So for this post I will use HTML + CSS (+ JS?) as an example for metalearning.

Research

Metalearning research can broadly be broken down into asking why, what and how.

Why?

There are two "types" of reasons for learning something. Intrinsic and Instrumental

Intrinsic motivation means the motivation comes from the sake of it

Instrumental motivation means the motivation comes from trying to achieve some nonlearning result.

Since my goal is to learn for the sake of improving my website, I have an instrumental motivation.

What?

Answering what is a matter of listing the concepts, facts and procedures.

I have decided to focus on concepts here because I think "facts" are just implementations of concepts so itd be silly to list all HTML Tags or whatever when the broad "HTML Tags" concept is enough.

Procedures are also omitted because the procedures include creating a HTML document and learning to push it to the server which is stuff I already know.

    Concepts:
  • HTML Tags
  • HTML Attributes
  • Semantic HTML
  • HTML Forms
  • Block Elements
  • Inline Elements
  • CSS Syntax
  • CSS Properties
  • The Box Model
  • Flexbox
  • CSS Grid
  • Positioning
  • Responsive Design
  • Animation

How?

Ultralearning (the book) recommends 2 methods for executing "How". "Benchmarking" and "Emphasise/Exclude"

Benchmarking is all about finding how other people have learnt what you want to learn.

After you have a base cirriculum you can make changes

Emphasise/Exclude is about taking a benchmarked ciriculum and deciding what to focus on and what to cut/delay

For me I am going to use Scrimba Frontend Developer course as my primary resource and the MDN Docs as a secondary resource
